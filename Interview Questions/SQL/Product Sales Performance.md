**SOLUTION**

```sql
WITH TopProducts AS(
    SELECT ProductID, Category, Revenue,
    DENSE_RANK() OVER (PARTITION BY Category ORDER BY Revenue DESC) AS dense_rank
    FROM ProductSales
)
SELECT ProductID, Category, Revenue
FROM TopProducts
WHERE dense_rank <= 3;
```
